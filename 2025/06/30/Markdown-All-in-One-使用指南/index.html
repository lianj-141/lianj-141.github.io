🎯 快捷键大全

基本操作

功能 Mac 快捷键 Windows 快捷键 说明

侧边预览 Cmd+K V Ctrl+K V 在侧边打开预览

列表操作

功能 快捷键 说明
切换任务状态 Alt+C 切换 [ ] 和 [x]
有序列表 Ctrl+Shift+] 创建数字列表
无序列表 Ctrl+Shift+[ 创建项目符号列表

⚡ 实用功能演示

1. 表格编辑增强

在表格中时，按 Tab 键可以：

  • 移动到下一个单元格
  • 自动对齐表格格式
  • 在最后一行时自动添加新行

2. 任务列表管理

使用 Alt+C 快速切换任务状态：

  • 完成的任务
  • 待完成的任务
  • 另一个完成的任务
  • 新的任务

3. 自动链接补全

输入 []() 时会自动补全，支持：

  • 文件路径补全
  • 标题锚点补全
  • URL 补全

4. 数学公式支持

行内公式：$a^2 + b^2 = c^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

5. 目录生成

使用命令面板 (Cmd+Shift+P) 搜索：

  • Markdown All in One: Create Table of Contents
  • 自动生成文档目录

📝 编辑技巧

快速格式化

  1. 选中文本 → 按快捷键应用格式
  2. 双击单词 → 按 Cmd+B 加粗整个单词
  3. 多行选择 → Alt+Click 多光标编辑

列表缩进

  • 按 Tab 增加缩进
    • 子项目会自动缩进
      • 支持多级缩进
  • 按 Shift+Tab 减少缩进

🔧 故障排除

常见问题

  1. 预览不更新：重启 VS Code 或重新加载窗口
  2. 表格格式混乱：使用 Shift+Alt+F 格式化
  3. 快捷键冲突：检查键盘映射设置
